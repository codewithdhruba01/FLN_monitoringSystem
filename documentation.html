Project Overview

The Daily Monitoring System for the FLN Program (Foundational Literacy and Numeracy) tracks classroom activities and student learning across all schools under the Halol and Maval clusters.

Purpose of the System

Subject Tracking

Monitor daily lessons, subjects, topics, and subtopics taught in each classroom.

Student Performance

Record attendance and learning indicators to identify students needing additional support.

Teacher Implementation

Track the use of TLM (Teaching Learning Materials) and FLN activity implementation.

Long-term Analytics

Enable data-driven insights through weekly, monthly, and quarterly progress tracking.

System Flow

The application follows a structured three-step navigation process ensuring accurate data context.

1

Cluster Selection

Users start by selecting their operational cluster (Halol or Maval). This filters the list of available schools.

2

School Selection

Users select their specific school from the dynamically filtered list based on the chosen cluster.

3

Teacher Dashboard

Teachers access their unique dashboard consisting of three core modules: Classes Held, Student Attendance, and Performance Monitoring.

Module 1 – Classes Held

Records daily activities across three main class types: FLN Classes, Remedial Classes, and Library Sessions.

FLN Class Details

  • Literacy Focus: Awareness, Reading, Writing
  • Numeracy Focus: Concepts, Operations, Shapes
  • Duration: 30 / 45 / 60 minute intervals
  • TLM Usage: Flashcards, Activity Kits, etc.

Remedial & Library Sessions

Focused sessions for storytelling, reading practice, and remedial support for core subjects like Mathematics and English.

  • Learning Check: Retrieval & retell capability
  • Feedback: Overall student engagement level
  • Reflection: Teacher's daily class experience

Module 2 – Student Tracking

Automatic loading of student lists for the selected school and class to record attendance and performance.

Feature Indicators
Attendance Present / Absent
Performance Rating Rating Scale 1 to 5
Learning Indicator Not Yet / With Help / Independently

Focus Student Tracking

Teachers select 5 students per day (rotating weekly) for deeper learning analysis and individual progression monitoring.

Module 3 – Teacher Monitoring

Capturing teacher-level implementation indicators and professional reflection.

  • Average time spent on FLN instruction
  • TLM utilization and effectiveness
  • Coverage of Literacy Categories
  • Coverage of Numeracy Categories
  • Level of student participation (Low/High)
  • Teacher reflection notes

Data Usage & Analytics

The collected data is stored in a central master database for high-level visual monitoring and decision-making.

Trend Analysis

Identification of attendance trends and school/cluster level comparisons.

Progress Reports

Generation of quarterly learning progress reports for stakeholders.
